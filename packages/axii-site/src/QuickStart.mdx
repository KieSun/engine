# 快速开始

## 创建应用

Use npx & degit:
```bash
npx degit ariesate/engine/packages/vite-axii my-project
cd my-project

npm install
npm run dev
```



## 官方组件/图标库

```bash
npm install axii-components --save
npm install axii-icons --save
```
