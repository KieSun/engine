# Quick Start

以 axii-components 作为示例。先确保在

- engine
- controller-axii
- axii-components
  下都执行了 `npm i`。鉴于速度问题，建议使用淘宝 npm 镜像，在 .bashrc/.zshrc 中 创建 alias：

```bash
alias taonpm="npm --registry=https://registry.npm.taobao.org"
```
